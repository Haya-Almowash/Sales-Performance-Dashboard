# Sales Performance Dashboard

This project focuses on analyzing sales performance using Excel and Power BI.

The dashboard provides insights into:
- Net revenue
- Customer types
- Regional sales performance
- Discount analysis
- Sales channels

The goal of this project is to help businesses track KPIs and understand sales trends through interactive visualizations.

## Tools Used
- Excel
- Power BI

## Project Files
- Dashboard → Power BI dashboard file
- Excel → Excel analysis file
- Images → Dashboard preview images
